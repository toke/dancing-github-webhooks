# What is this?

I use this as an Webhook endpoint for getting notified by [Github](https://github.com/)
when a repository is changed.

[Read more](http://toke.de/blog/perl/2012/02/09/how-i-post/)

## Installation

     cpan Dancer
     perl bin/app.pl

## Deployment
I use Perlbal...
`cpan IO::AIO Perlbal`
